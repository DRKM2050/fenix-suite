Credenciales API Personalizadas
Guía para configurar su Nube (Google Cloud):
  1. Acceda a Google Cloud Console e inicie sesión.
  2. Cree un proyecto nuevo (ej: FENIX Suite Sync).
  3. Vaya a la Biblioteca de APIs y active la API llamada Google Drive API.
  4. Configure la Pantalla de consentimiento de OAuth (Tipo de usuario: Externo, ingrese datos de soporte y guarde).
  5. Paso Crítico: En la sección Usuarios de prueba (Test Users) agregue el correo de Google con el que iniciará sesión.
  6. Vaya a Credenciales > Crear credenciales > ID de cliente de OAuth. Elija tipo de aplicación: Aplicación de escritorio (Desktop App).
  7. Copie el Client ID y Client Secret generados, colóquelos abajo, haga clic en Guardar Credenciales y luego conecte su cuenta.
