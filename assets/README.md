# Directorio de Assets (FNX-ADMIN)

Coloque en esta carpeta los recursos gráficos y multimedia requeridos por la aplicación:

1. `icon.ico` - Icono de la aplicación para empaquetado en Windows.
2. `icon.png` - Icono de la aplicación en alta resolución para macOS y Linux.
3. Logotipos e imágenes estáticas del sistema.
